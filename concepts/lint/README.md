
Use `ast-grep` to analyze past style to get style!

1. ES Module imports (import not require)
2. Proper copyright headers with attribution
3. Named functions for main operations, arrow functions for inline callbacks
4. Semicolons at end of statements
5. Destructuring for imports when appropriate
6. const/let over var
7. Single quotes for strings (mostly)
8. Proper spacing in object/array literals
